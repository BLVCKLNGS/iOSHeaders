# iOSHeaders

List all the iOS Headers from `5.1` to `12.1.2`

This project is using:
- VueJS
- Typescript
- NodeJS

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
